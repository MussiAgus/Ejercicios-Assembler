# Laboratorio de Arquitectura de Computadoras (RISC-V)

Repositorio dedicado al estudio de la arquitectura de computadoras y programación en bajo nivel utilizando el conjunto de instrucciones **RISC-V (RV32I)**. Las soluciones aquí presentadas forman parte de la guía práctica de la **Universidad Nacional de La Matanza (UNLaM)**.

## Objetivos del Proyecto
- Comprender el funcionamiento interno de un procesador RISC.
- Gestionar manualmente recursos de hardware (registros, memoria y stack).
- Desarrollar algoritmos eficientes en lenguaje ensamblador.
- Analizar el comportamiento del **Pipelining** y la resolución de conflictos (Hazards).

## Estructura del Repositorio
El proyecto está organizado en 4 módulos principales:

1. Funciones con Arrays : Ordenamiento (Bubble Sort), búsqueda de máximos/mínimos y manipulación de vectores.
2. Funciones con Strings : Inversión de cadenas, cálculo de longitud y búsqueda de patrones.
3. Funciones Conocidas : Reimplementación de librerías estándar como atoi, toLower y comparaciones case-insensitive.
4. Operaciones Aritméticas : Multiplicación y división optimizada mediante bit-shifting.

## Herramientas
- **Simulador:** [Ripes](https://github.com/mortbopet/Ripes) (Visualización de Data Path y Pipeline).
- **ISA:** RISC-V RV32I (Base Integer Instruction Set).
